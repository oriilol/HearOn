# HearOn

HearOn is a modern, open-source music player for Android built entirely with Kotlin and Jetpack Compose. It offers a seamless streaming experience, automatic offline caching, and a highly customizable user interface based on Material Design 3.

## Features

* **Material Expressive Design:** Fully adapts to your system theme with dynamic color extraction from album covers.
* **Smart Playback Engine:** Gapless playback and adjustable crossfade between tracks.
* **Offline Support:** Tracks are automatically cached locally after the first play to save data on subsequent listens.
* **Synchronized Lyrics:** View real-time synced lyrics for your favorite tracks.
* **Library Management:** Create playlists, save favorite tracks, and customize track covers.
* **Adaptive Layouts:** Fully responsive UI that scales beautifully on both smartphones and tablets (landscape and portrait).
* **Smart Recommendations:** Auto-generated queue based on your listening history.

## Installation

You can download the latest APK from the [Releases](https://github.com/oriilol/hearon/releases) page.

## Building from source

1. Clone the repository: `git clone https://github.com/oriilol/hearon.git`
2. Open the project in Android Studio (Koala or newer recommended).
3. Sync the Gradle project and build the APK.

## Known Issues and Bug Reporting

If you find a bug, please check the existing issues or open a new one in the Issues tab with steps to reproduce it.
